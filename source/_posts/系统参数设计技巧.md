## 1.controller入参设计
#### 请求基础类
```java
@Getter
@Setter
public abstract class BaseRequest implements Serializable {

    @Serial
    private static final long serialVersionUID = 3182217732056533200L;

    private String traceId;

    private String clientVersion;

    private Long timestamp;


    @NotNull(message = "traceId 不能为空")
    @Pattern(regexp = "^[a-zA-Z0-9]{32}$", message = "traceId 格式错误")
    public String traceId() {
        return traceId;
    }


    public abstract void validate();

}

```

#### 分页请求参数基础类
```java
@Getter
@Setter
public abstract class PageRequest extends BaseRequest{

    @Min(value = 1, message = "page 不能小于 1")
    private int page;

    @Max(value = 100, message = "pageSize 不能大于 100")
    private int pageSize;

    private String sortFiled;

    private SortType sortType;


    @Override
    public void validate() {

    }
}

```

排序枚举
```java
public enum SortType {

    ASC,

    DESC;
}
```


#### 分页参数包装类
```java
@Getter
@Setter
@ToString
public class PageParam<T> extends PageRequest{

    private T data;

}

```

#### 业务请求参数
```java
@Getter
@Setter
@ToString
public class LiconRequestParam extends BaseRequest {

    private String username;

    private String password;

    @Override
    public void validate() {
        if (!StringUtils.hasText(username)){
            throw new RuntimeException("非法的用户名");
        }

        if (!StringUtils.hasText(username)){
            throw new RuntimeException("非法的密码");
        }
    }
}
```

## 2.多类型业务入参设计

> 设计的好处,将请求类型与响应类型相绑定，不需要每次执行完逻辑后都需要指定相应的类型 </br>
> 而是直接使用请求上所绑定的类型进行返回，后面就算新增业务场景，也无需修改客户端，只需要添加请求和响应即可

#### 请求基类
```java

public abstract class BaseRequest <T extends BaseResponse> implements Serializable {

    public Class<T> responseType;

    @SuppressWarnings("unchecked")
    public Class<T> getResponseType() {
        if (responseType == null){
            Type genericSuperclass = getClass().getGenericSuperclass();
            responseType = (Class<T>) ((ParameterizedType) genericSuperclass).getActualTypeArguments()[0];
        }
        return  responseType;
    }
}

```

#### 响应基类
```java
public abstract class BaseResponse implements Serializable {

}
```

#### 客户端业务处理类
```java
public class BizClient {

    public static <T extends BaseRequest<R>,R extends BaseResponse> R execute(T request){
        return JSON.copyTo(request,request.getResponseType());
    }
}
```


#### 业务请求类
```java
@EqualsAndHashCode(callSuper = true)
@Data
@Accessors(chain = true)
public class LiconRequest extends BaseRequest<LiconResponse>{

    private String username;

    private String password;

    private String reqType = "request";

}
```

#### 业务响应类
```java
@EqualsAndHashCode(callSuper = true)
@Data
public class LiconResponse extends BaseResponse{

    private String username;

    private String password;

    private String resType = "response";
}
```

#### 使用示例
```java
        LiconResponse res = BizClient.execute(new LiconRequest()
                .setUsername("Licon")
                .setPassword("123456"));

        System.out.println(res);
```
