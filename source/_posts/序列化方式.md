#### 主流序列化对比

| 特性                | Kryo               | Protobuf           | Thrift             | JSON               |
 |---------------------|--------------------|--------------------|--------------------|--------------------|
| **序列化速度**      | 非常快（Java最优）| 非常快             | 非常快             | 慢                 |
| **反序列化速度**    | 非常快             | 非常快             | 非常快             | 慢                 |
| **数据大小**        | 较小               | 非常小             | 非常小             | 大                 |
| **跨语言支持**      | 仅Java             | 广泛               | 广泛               | 非常广泛           |
| **模式演进支持**    | 弱                 | 强                 | 强                 | 中（需应用层处理） |
| **易用性**          | 简单（Java）       | 中等（需定义模式） | 中等（需定义模式） | 非常简单           |
| **人类可读**        | 否                 | 否                 | 否（默认）         | 是                 |
| **适用场景**        | Java内部通信       | 跨语言微服务、存储 | 跨语言微服务、RPC  | Web API、配置文件  |
| **RPC支持**         | 无                 | 有（gRPC）         | 有（Thrift RPC）   | 无（通常基于HTTP） |


#### kryo

引入kryo依赖
```xml
<dependency>
    <groupId>com.esotericsoftware</groupId>
    <artifactId>kryo</artifactId>
    <version>5.6.2</version>
</dependency>
```

对应序列化与反序列化代码
```java
public static void main(String[] args) throws Exception{
        Kryo kryo = new Kryo();
        kryo.register(User.class);
        
        //一个普通的User对象
        User user = User.builder()
                .id(1)
                .name("张三")
                .email("zhangsan@163.com")
                .phone("12345678901")
                .address("中国")
                .birthday(LocalDate.now().toString())
                .sex(1)
                .avatar("https://avatar.com/avatar.png")
                .build();
        
        //序列化
        try (FileOutputStream fos = new FileOutputStream("user.txt");
            Output output = new Output(fos)){
            kryo.writeObject(output, user);
        }

        //反序列化
        try(FileInputStream fis = new FileInputStream("user.txt");
            Input input = new Input(fis)){
            User readUser = kryo.readObject(input, User.class);
            System.out.println(readUser);
        }

}
```

#### protobuf

需要下载protoc的编译器

编写.proto文件

```proto
syntax = "proto3";
package com.ccb.protobuf;

option java_package = "com.ccb.protobuf";
option java_outer_classname = "UserProto";
option optimize_for = SPEED;
option java_multiple_files = true;


message User {
  int32 id = 1;
  string name = 2;
  string email = 3;
  string phone = 4;
  string address = 5;
  string birthday = 6;
  int32 sex = 7;
  string avatar = 8;
}
```


使用protoc编译器生成对应的java文件或者c文件，python文件等
```cmd
protoc --java_out=. user.proto
```
例如：java

- User.java
- UserOrBuilder.java
- UserProto.java

引入相关依赖，然后使用生成的文件即可
```xml
<!--protobuf java版本-->
<dependency>
    <groupId>com.google.protobuf</groupId>
    <artifactId>protobuf-java</artifactId>
    <version>4.31.1</version>
</dependency>

<!--用于json和protobuf的转换-->
<dependency>
    <groupId>com.google.protobuf</groupId>
    <artifactId>protobuf-java-util</artifactId>
    <version>4.31.1</version>
</dependency>
```

使用示例
```java
public static void main(String[] args) throws Exception {
        User build = User.newBuilder()
                .setId(1)
                .setName("张三")
                .setEmail("zhangsan@163.com")
                .setPhone("12345678901")
                .setAddress("中国")
                .setBirthday(LocalDate.now().toString())
                .setSex(1)
                .setAvatar("https://avatar.com/avatar.png")
                .build();
                
        //序列化
        try (FileOutputStream output = new FileOutputStream("user.data")) {
            build.writeTo(output);
        }
}
```

#### Thrift

下载 Apache Thrift 编译器

编写.thrift文件
```text
namespace java com.ccb.thrift

// 用户结构体
struct User {
    1: required i32 id,
    2: required string name,
    3: optional string email,
    4: optional list<string> phones
}

// 用户服务接口
service UserService {
    // 获取用户信息
    User getUser(1: i32 userId),

    // 添加用户
    bool addUser(1: User user),

    // 获取所有用户
    list<User> listUsers(),

    // 搜索用户
    list<User> searchUsers(1: string keyword)
}
```

使用编译器编译.thrift文件，生成对应的代码：
```text
thrift --gen java user_service.thrift
```

引入对应依赖后即可使用生成的java文件

- User.java
- UserService.java


```xml
<dependency>
    <groupId>org.apache.thrift</groupId>
    <artifactId>libthrift</artifactId>
    <version>0.22.0</version>
</dependency>
```

编写实现类：
```java
public class UserServiceImpl implements UserService.Iface{
    @Override
    public User getUser(int userId) throws TException {
        return new User(userId, "ccb");
    }

    @Override
    public boolean addUser(User user) throws TException {
        System.out.println("add user: " + user);
        return true;
    }

    @Override
    public List<User> listUsers() throws TException {
        return List.of(new User(1, "List Licon User"));
    }

    @Override
    public List<User> searchUsers(String keyword) throws TException {
        return List.of(new User(1, "Search Licon User"));
    }
}
```

启动服务端：

```java
public static void main(String[] args) throws Exception {
    // 创建用户服务处理器实例
    UserService.Processor<UserServiceImpl> userServiceProcessor = new UserService.Processor<>(new UserServiceImpl());

    // 创建服务器套接字，监听8080端口
    TServerSocket tServerSocket = new TServerSocket(8080);

    // 使用线程池服务器，并配置处理器和服务器套接字
    TThreadPoolServer tThreadPoolServer = new TThreadPoolServer(new TThreadPoolServer.Args(tServerSocket).processor(userServiceProcessor));
    
    // 启动服务器，开始提供服务
    tThreadPoolServer.serve();
}

```

启动客户端：

```java
public static void main(String[] args) throws Exception{
    // 创建一个TSocket实例，指定主机为localhost，端口为8080
    TSocket socket = new TSocket("localhost", 8080);
    // 打开socket连接
    socket.open();

    // 创建一个TBinaryProtocol实例，使用之前打开的socket进行通信
    TBinaryProtocol tBinaryProtocol = new TBinaryProtocol(socket);
    
    // 创建UserService的客户端实例，使用二进制协议进行数据传输
    UserService.Client client = new UserService.Client(tBinaryProtocol);
    
    // 通过客户端实例调用getUser方法，传入用户ID为1，获取用户信息
    User user = client.getUser(1);

    // 打印获取到的用户信息
    System.out.println(user);
}
```
